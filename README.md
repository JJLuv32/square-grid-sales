# Square Grid Sales

A React application for selling squares on a 10x10 grid with Venmo and Cash App payment integration.

## Features

- Interactive 10x10 grid (100 squares)
- $5 per square
- Add initials to selected squares
- Real-time running total
- Direct Venmo payment link (@Jonathan-Coleman-80)
- Direct Cash App payment link ($jjluv)
- Responsive design

## Payment Setup

When users click "Pay with Venmo" or "Pay with Cash App":
- Selected squares are automatically locked
- Payment app opens with pre-filled amount and note
- User completes payment in their app
